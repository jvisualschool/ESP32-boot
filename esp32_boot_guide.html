ESP32-S3
BOOT JOURNEY

전원을 켜는 순간부터 당신의 코드가 실행되기까지의 260ms, 그 경이로운 여정을 완벽하게 해석합니다.

1. 부팅이란 무엇인가?

부팅(Booting)은 컴퓨터가 전원을 켜고 사용 가능한 상태가 되기까지의 모든 준비 과정을 말합니다. ESP32-S3는 단 0.26초 만에 이 모든 일을 끝내죠.

아침 루틴 비유

알람 소리 듣기 전원 인가
눈 뜨기 ROM 부트로더 시작
화장실 가기 하드웨어 초기화
옷장에서 옷 꺼내기 메모리 로딩
출근 준비 완료 애플리케이션 실행

성능 지표

총 소요 시간
260ms
전력 효율
최상

윈도우 PC가 30초 걸릴 때, ESP32는 그보다 약 115배나 더 빠르게 기상합니다.

2. 실시간 로그 분석기

시리얼 모니터에 찍히는 외계어 같은 로그들, 이제 한 줄씩 클릭해서 그 의미를 파헤쳐 보세요.

esp32s3_serial_monitor.log

왼쪽의 로그 라인을 클릭하면
상세 설명이 나타납니다.

ROM BOOTLOADER

ESP-ROM:esp32s3-20210327

이 칩의 ROM은 2021년 3월 27일에 만들어졌습니다. 칩 제조 시 내장되어 절대 변경되지 않는 DNA와 같은 영역입니다.
비유: 사람의 지문처럼 칩이 태어날 때부터 가지고 있는 고유 정보입니다.

3. 메모리 세그먼트 (Image Segments)

프로그램은 단순한 하나의 덩어리가 아닙니다. 용도에 따라 여러 조각으로 나뉘어 책장(Flash)에서 책상(RAM)으로 옮겨집니다.

0x3C020020 Segment 0: 메인 코드 (Flash Map) 34 KB
0x3FC92000 Segment 1: 데이터 변수 (DRAM) 10 KB
0x40374000 Segment 2: 고속 코드 (IRAM) 20 KB
0x42000020 Segment 3: 읽기 전용 (Flash Read) 88 KB
0x50000000 Segment 5: RTC 메모리 32 Bytes
