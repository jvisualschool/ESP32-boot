# ESP32-S3 부팅 과정 완벽 가이드 (ESP32-S3 Boot Master)

초보자도 쉽게 이해할 수 있는 **ESP32-S3 부팅 프로세스 인터랙티브 가이드**입니다.
전원이 켜지는 순간부터 내 코드가 실행되기까지 칩 내부에서 일어나는 모든 일을 시각적으로 설명합니다.

## 📂 파일 구성

- **index.html**: 메인 가이드 페이지 (웹 브라우저에서 실행)
- **부팅.md**: 부팅 과정에 대한 상세 기술 문서 (원본 데이터)

## ✨ 주요 특징

1.  **단계별 심층 분석**
    - **ROM 부트로더**: 칩에 내장된 불변의 첫 번째 코드
    - **2nd Stage 부트로더 (ESP-IDF)**: 하드웨어 초기화 및 보안 설정
    - **파티션 테이블**: Flash 메모리 구조와 NVS, Factory 파티션
    - **세그먼트 로딩**: Flash에서 RAM으로 데이터가 이동하는 과정 (Load vs Map)
    - **앱 시작**: `app_main()`이 호출되기까지의 과정

2.  **인터랙티브 학습 요소**
    - **퀴즈**: 각 챕터마다 핵심 내용을 확인하는 퀴즈 제공
    - **로그 분석**: 실제 터미널 로그를 한 줄씩 분석하여 의미 설명
    - **시각화**: 메모리 맵(Memory Map), 부팅 타임라인 시각화

3.  **편의 기능**
    - **용어 사전**: UART, GPIO, Flash, Heap 등 필수 임베디드 용어 정리
    - **문제 해결 가이드**: 무한 재부팅(Boot Loop), 침묵(Silent Boot) 등 흔한 문제 해결법
    - **전체 로그 보기**: 실제 부팅 로그 전문을 한눈에 확인 가능

## 🚀 사용 방법

1. `index.html` 파일을 크롬, 엣지, 사파리 등 모던 웹 브라우저로 엽니다.
2. 사이드바의 목차를 따라 순서대로 학습하거나, 궁금한 부분을 찾아봅니다.
3. 중간중간 나오는 퀴즈를 풀며 이해도를 점검합니다.
